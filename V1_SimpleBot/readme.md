### V1 Simple Bot [OpenAI, LLamaIndex]

1. create project folder
2. (optional) create readme.md
   1. `ni readme.md`
3. create virtual environment
   1. `python -m venv .venv`
4. Activate virtual env
   1. `.venv/Scripts/activate`
5. create requirements.txt and
   1. `ni requirements.txt`
6. install required libraries
   1. `python -m pip install -r requirements.txt`
7. `setup .venv kernel for Jupyter Notebook`
8. Create .env file for credentials/secrets
   1. `ni .env`
   2. store OPEN_AI_API key in .env file
